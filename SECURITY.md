Currently supported versions of OnpointDoc:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of OnpointDoc seriously. If you discover a security vulnerability, please follow these steps:

### How to Report

**Email**: support@onpointafrica.com

**Subject**: [SECURITY] Vulnerability Report - OnpointDoc

**Please include**:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### What to Expect

- **Response time**: Within 48 hours
- **Updates**: We'll keep you informed of our progress
- **Fix timeline**: Critical issues within 7 days, others within 30 days
- **Credit**: We'll acknowledge your contribution (if desired)

### Responsible Disclosure

Please:
- ✅ Give us reasonable time to fix the issue before public disclosure
- ✅ Avoid exploiting the vulnerability beyond proof-of-concept
- ✅ Don't access or modify data that isn't yours

Thank you for helping keep OnpointDoc secure!
